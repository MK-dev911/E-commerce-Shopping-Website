# 🌸 Responsive E-Commerce Website

A modern, responsive e-commerce storefront for a fashion boutique. Built from scratch using HTML5, CSS3, and modern Vanilla JavaScript. The website features a dynamic shopping cart, a wishlist system, and smooth sliding interactive panels.

## 🚀 Live Demo
*   Main Storefront: `index.html`
*   Bags Category Page: `bags.html`
*   Accessories Category Page: `accessories.html`

---

## 🔥 Key Features

*   **Dynamic Shopping Cart:** Add items to the cart, adjust quantities, and watch totals update automatically without reloading the page.
*   **Wishlist System:** Save favorite items with a heart toggle button and view them in a separate wishlist area.
*   **Interactive Sidebar Drawers:** Smooth sliding slide-out panels for Search, User Login, Shopping Cart, and Wishlist.
*   **Animated Menu Toggle:** A minimalist hamburger icon that dynamically morphs into an "X" close icon on mobile displays.
*   **Fully Responsive:** Optimized layout patterns matching all screen viewports (Desktop, Tablet, and Mobile devices).

## 🛠️ How to Run the Project

You do not need to install any databases, servers, or packages to run this project! Follow these simple steps:

### Option 1: Direct Run (Easiest)
1. **Download or clone** this repository to your computer.
2. Locate the project folder and **double-click `index.html`**. 
3. It will instantly open and run perfectly inside your default web browser (Chrome, Safari, Edge, etc.).

### Option 2: Using VS Code Live Server (Best for Editing)
1. Open the project folder inside **Visual Studio Code**.
2. If you don't have it, install the **Live Server** extension (by Ritwick Dey).
3. Right-click on `index.html` in your sidebar and select **Open with Live Server**.
4. The site will open in your browser and automatically refresh every time you save a code change!
